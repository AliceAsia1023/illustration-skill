# Examples

Use these as reference patterns when the user asks for a directly generatable prompt.

## Network Access Routing

Use when the concept is about opposite-direction access restrictions, relays, regional blocks, or rerouting through a host in another location.

### Example Idea

Some foreign websites are blocked from mainland China, but can be reached through a Hong Kong host. Some top AI tools still block mainland China and Hong Kong, so access must continue through another overseas host such as Singapore.

### Example Prompt

```text
Create one 16:9 horizontal editorial illustration on a pure white background using the $girl-article-illustrations style. It must look like a sparse black-marker doodle series image: thin to medium uneven black marker lines, lots of whitespace, no shading, no gradients, no texture, no polished finish. Handwritten Chinese notes should be few and short.

Girl style: a very small secondary twin-tail doodle girl, round head, dot eyes, tiny mouth, scribbly black hair, two simple ponytails, single-stroke limbs, awkward proportions, no anime features. She is an operator, not the protagonist.

Core idea: a Hong Kong host acts as a key relay station. From mainland China, a Hong Kong host can help access normal foreign websites, but for top AI tools, even Hong Kong is still blocked, so traffic must continue detouring through another overseas node such as Singapore.

Scene: draw a simple network relay structure as the main subject. On the left, a box labeled "中国内地". In the center, make a larger and more visually important relay box labeled "香港主机". On the upper right, a destination cluster labeled "外网" with tiny notes like "YouTube" "GitHub" "Gmail" "Google"; show an arrow path from mainland to Hong Kong host to foreign websites, with a note like "可访问". On the lower right, another destination cluster labeled "AI工具" with tiny notes like "Codex" "Claude"; show a blocked mark after the Hong Kong host, then a detour path through a small extra node labeled "新加坡主机" before reaching AI tools. The Hong Kong host must be visually central and clearly function as the relay hub for both paths. The small twin-tail doodle girl is actively plugging, rerouting, or dragging the cable path around the Hong Kong host and onward through Singapore.

Add only short handwritten Chinese labels, around 6 to 8 total: "中国内地", "香港主机", "外网", "可访问", "AI工具", "仍受限", "新加坡主机", "绕路". Use mostly black linework, a few orange path arrows, and at most one tiny red blocked mark.

The image should feel like a quick explanatory editorial sketch, not a poster, not a product diagram, not a cute illustration.

Avoid: anime line art, polished illustration, cute girl, mascot feel, children-book style, infographic layout, glossy drawing, dense text, decorative-only character.
```
