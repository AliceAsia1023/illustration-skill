# Series Lock

Use this file when one article needs multiple matched illustrations.

## Goal

Keep the full set feeling like one editorial doodle series instead of unrelated single images.

## Non-Negotiable Series Consistency

- Same pure white background
- Same sparse black-marker doodle language
- Same thin-to-medium uneven black line weight
- Same twin-tail girl style and scale
- Same broad whitespace density
- Same restraint in handwritten Chinese notes
- Same restrained orange/red/blue accent usage

## Girl Scale

The girl should stay secondary in all images.

- She is an operator, not the main subject
- She should usually be smaller than the main structure
- Do not let one image become character-led while others remain structure-led

## Annotation Density

Keep labels visually consistent across the set.

- Usually 4 to 5 short handwritten Chinese labels per image
- Keep label size similar across the series
- Avoid one image becoming text-heavy

## Composition Discipline

- One image, one idea
- One dominant structure per image
- Leave one visibly empty white area in each frame
- Do not over-detail buildings, devices, or props

## Proven Article Workflow

For article adaptation, default to:

1. extract 3 to 5 anchors
2. map one anchor to one image
3. produce a shot list
4. write prompts
5. if generating, refine the whole set for series consistency

## Reference Prompt Add-On

When generating a series, add wording like:

`It must match the same series style: sparse black-marker doodle, same line weight, same small secondary twin-tail girl scale, lots of whitespace, same annotation density.`
